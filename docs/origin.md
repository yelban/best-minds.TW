# 發想來源

## Andrej Karpathy 的 LLM 模擬器觀點

這個 Skill 的核心理念源自 Andrej Karpathy 在 X 上的一則推文，獲得超過 27k 讚、2.8k 轉發。

### 核心論點

1. **LLM 是模擬器，不是實體**
   - 不要把 LLM 當成「有想法的實體」（entities）
   - 它是「模擬器」（simulators），能呈現任何人的思維方式

2. **沒有真正的「你」**
   - LLM 沒有自己形成的觀點
   - 當你問「你怎麼看」時，實際上是強迫它採用 finetuning 資料的統計平均人格
   - 得到的是某種「平均觀點」，而非真正有洞見的回答

3. **正確的使用方式**
   - 指定你想模擬的視角或人物
   - 讓 LLM 扮演該領域最頂尖的專家
   - 這樣比問「你怎麼看」更能發揮 LLM 的能力

### 原文引用

> — Andrej Karpathy
