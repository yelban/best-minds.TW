# Best Minds Skill 使用指南

## 能力說明

這是一個**思維方法論** skill，改變 AI 回答問題的方式：

| 傳統問法 | Best Minds 問法 |
|---------|----------------|
| 「你怎麼看？」 | 「世界上誰最懂這個？TA 會怎麼說？」 |
| AI 用自己視角回答 | AI 模擬頂級專家回答 |

## 觸發詞

在對話中使用以下詞彙即可觸發：

- 最強大腦
- 頂級專家
- 世界級
- best minds
- 誰最懂這個

## 使用範例

```
「關於量子計算的未來，誰最懂這個？」
→ AI 會找出該領域最強專家（如 John Preskill），模擬其觀點回答

「我想創業，請用頂級專家的視角給建議」
→ AI 可能模擬 Paul Graham、Marc Andreessen 等人
```

## 與 ai-coaches 的差異

| ai-coaches | best-minds |
|------------|------------|
| 從 13 個預設智者選 | 從全世界找最懂的人 |
| 基於關鍵詞匹配 | 基於問題本質 |

## 核心價值

利用 LLM 作為「模擬器」的特性——它訓練資料包含大量專家的著作、演講、訪談，所以能合理模擬這些人的思維方式和觀點。

> "Don't think of LLMs as entities but as simulators."
> — Andrej Karpathy
